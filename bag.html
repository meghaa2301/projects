Zipper Bags / MAGIC SEAL® Bags / ZIP LOCK® Bags / SELF LOCK® Bags


Zipper bags (ZIP LOCK® bags) which are made out of Low density polyethylene ( LDPE ) with an inbuilt press to close zipper profile on the bag to make the bag re-closable.


This unique reclosable feature adds an element of convenience for the end user by making it reusable and recyclable.


Zipper bags :
  • Plain transparent Zipper bags / ZIP LOCK® bags
  • Printed zipper bags upto 8 colors.
  • Zipper bags with Redline on the flap above zipper
  • Zipper bags with coloured line on the flap above zipper ( Any different colour )
  • WOP Zipper bags ( Write on panel )
  • Double Zip - Zipper bags (Bags with double zipper lock )
  • Coloured Zipper bags ( Milky white / Any other coloured film zipper bags )
  • 3 wall / 4 wall Kangaroo zipper bags
